# **원티드 프리온보딩코스**

원티드에서 진행하는 프리온보딩코스 지원 과제.

반응형 NavBar 만들기.

## **실행 방법**

### 설치

`npm install`

### 실행

`npm start`

## 구현 목록

`로그인 / 로그아웃`
- 로그인과 로그아웃 상태에 따라 bar 형태가 달라진다.

`반응형 웹`

- Mobile, Tablet, Desktop에 맞게 각각의 웹 디자인 설계.

`메뉴 hover`

- 메뉴에 마우스 over할 때, 부수적 menu 드롭다운.
